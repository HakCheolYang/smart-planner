# 스마트 플래너 변경 이력

---

## v1.3.2 (2026-06-25)

### 버그 수정
- **스크롤된 상태에서 블록을 드래그하면 다른 시간대로 튀는 문제**: 타임라인을 스크롤해 오후 블록을 본 뒤 드래그하면, 마우스 위치를 스크롤 안 된(맨 위) 기준으로 계산해 블록이 오전 쪽으로 잘못 이동하고 그 위치로 고정 시간이 저장되던 문제 수정
  - 드래그·리사이즈 위치 계산에 컨테이너의 스크롤 오프셋을 반영

---

## v1.3.1 (2026-06-25)

### 버그 수정
- **시작/종료 시간을 기록해도 블록 위치가 안 바뀌는 문제**: 실제 시작·종료 시각(`actStarts`/`actEnds`)은 저장되지만 타임라인 블록은 원래 예정된 시간(고정 시간 등)에 그대로 남아있던 문제 수정
  - 그날 블록이 하나뿐인 태스크는 시작 시각을 기록하면 블록이 실제 시각 위치로 옮겨지고 `🕐실제시간` 표시가 붙음
  - 종료 시각까지 기록하면 블록 길이도 실제 소요 시간에 맞게 조정됨
- **드래그가 화면에 보이는 범위(주로 오전) 밖으로 안 움직이던 문제**: 타임라인이 스크롤 영역인데 드래그/리사이즈 중 자동 스크롤이 없어, 보이지 않는 시간대(오후 등)로는 끌어다 놓을 수 없던 문제 수정
  - 드래그/리사이즈 중 마우스가 위/아래 가장자리에 가까워지면 자동으로 스크롤됨

## v1.3.0 (2026-06-25)

### 버그 수정
- **완료해도 타임라인에서 사라지는 문제**: `taskActive`가 완료(`done`)된 태스크를 무조건 제외시켜, 완료 버튼을 누르면 그날 블록까지 통째로 사라지던 문제 수정
  - 완료된 태스크도 해당 날짜 블록엔 남아 체크 표시·반투명 처리로 보이고, 다른 날엔 기존처럼 나타나지 않음
  - 반복 태스크의 날짜별 완료 여부(`doneDates`)도 타임라인에 정확히 반영되도록 통일 (`isDoneOn`)
- **마감일 D-day 배지가 하루 더 표시되는 문제**: 마감일 문자열을 UTC로 해석해 한국 시간대(UTC+9)에서 항상 하루 지난 것처럼 계산되던 문제 수정 (`dleft`)

### 신규 기능
- **실제 시작/종료 시간 기록**: 일간 타임라인 블록에 ▶시작 → ⏹종료 2단계 버튼 추가
  - 클릭 시 시간 선택 모달이 열리고, 5·10·15·30분 전/후 칩 또는 직접 입력으로 실제 시각을 보정할 수 있음
  - 종료를 확정하면 완료 처리되며, 블록에 실제 시작~종료 시각이 표시됨
- **미착수 표시**: 예정 시간이 지났는데 시작하지 않은 블록에 점선 테두리와 `⚠️미착수` 배지 표시
- **반복 태스크 회차별 수정**: 반복 태스크를 수정(✏️)하면 "이번 회차만 수정" / "전체 반복 일정 수정" 선택 팝업이 뜸
  - 이번 회차만 수정을 고르면 반복 옵션은 숨겨지고, 원본 반복 일정은 그대로 유지된 채 해당 날짜 전용 사본이 별도로 생성됨
- **메모 필드**: 모든 태스크에 자유 메모를 입력할 수 있고, 태스크 카드와 타임라인 툴팁에 표시됨

## v1.2.2 (2026-06-24)

### 버그 수정
- **드래그로 옮긴 시간이 되돌아가는 문제**: 타임라인 블록을 드래그/리사이즈해도 화면에만 반영되고 실제 태스크에는 저장되지 않아, 체크박스 토글 등 다른 동작 후 원래 시간으로 되돌아가던 문제 수정
  - 드래그/리사이즈를 마치면 해당 태스크의 `⏰ 고정 시간`으로 실제 저장되어 새로고침해도 유지됨

---

## v1.2.1 (2026-06-24)

### 버그 수정
- **긴급 태스크가 과거 시간대에 배치되는 문제**: 긴급 삽입 시 항상 하루의 첫 시간대(아침)부터 배치되어, 오후에 삽입해도 이미 지나간 아침 슬롯에 꽂히던 문제 수정
  - 오늘 일정에서는 긴급 태스크가 현재 시각 이전 시간대를 건너뛰고, 현재 시각부터 배치됨
  - 뒤에 줄 서 있던 우선순위 낮은 태스크는 자동으로 밀리거나 자리가 없으면 오늘 일정에서 제외됨

## v1.2.0 (2026-06-24)

### 신규 기능
- **현재시간 화살표**: 일간 타임라인 시간축에 현재 시각을 가리키는 화살표 표시, 30초마다 자동 갱신
- **습관 트래커**: 반복 태스크(체크/고정/유연)의 완료 날짜를 기록해 연속 달성일(스트릭)을 계산
  - 체크리스트·태스크 카드에 `🔥N일 연속` 배지 표시
  - 분석 탭에 태스크별 최근 14일 히트맵 추가 (완료/미완료/해당없음 구분)
- **실제 활동 로그**: 일간 스케줄 하단에 그날 실제로 한 일을 시간과 함께 자유롭게 기록/삭제

### 개선
- 반복 태스크의 완료 기록을 단일 `lastDoneDate`/`completedCount`에서 날짜 배열(`doneDates`)로 변경해 히스토리 기반 통계가 가능해짐

## v1.1.0 (2026-06-24)

### 신규 기능
- **고정 시간 옵션 추가**: 태스크 추가/수정 폼의 허용 시간대 섹션에 `⏰ 고정 시간` 버튼 추가
  - 활성화 시 시간 입력창이 나타나며, 정확히 해당 시간에 타임라인 블록이 배치됨
  - 고정 시간 사용 시 허용 시간대 버튼은 자동 비활성화
  - 타임라인 블록에 `⏰고정` 표시, 태스크 카드에 `⏰ HH:MM 고정` 표시

### 개선
- 헤더에 버전 번호 표시 추가 (배포 확인 용이)

## v1.0.0 (2026-06-24)

### 최초 출시
- **스케줄 탭**: 일간 타임라인 / 주간 그리드 뷰, 블록 드래그로 시간 조정
- **태스크 탭**: 6가지 태스크 유형 (체크, 일정, 고정, 유연, 긴급, 위시)
- **분석 탭**: 카테고리별 시간, 목표별 투자 시간, 삶의 균형 점수
- **설정 탭**: 사명서, 인생 목표, 시간대, 뽀모도로 설정
- **뽀모도로 타이머**: 헤더 상시 표시, 세션 완료 시 브라우저 알림
- **반복 일정**: 매일/매주/매월/평일/주말 반복 지원
- **긴급 삽입**: 긴급 태스크 추가 시 유연 태스크 자동 압축
- **localStorage 자동 저장**: 새로고침해도 데이터 유지
